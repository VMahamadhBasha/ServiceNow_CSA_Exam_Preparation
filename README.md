PART 1 — Exam Analysis (2026 latest)
ServiceNow CSA (Certified System Administrator)

This is the foundation certification. It proves you understand the platform and can administer it.

Exam format

From the official blueprint:

Questions: 60
Duration: 90 minutes
Question type:
Multiple choice
Multi-select (choose 2 or more)
Result shown immediately after exam

Passing score

ServiceNow never publishes the exact cut score, but all official guides and prep providers align around ~70%.

Difficulty level

Beginner → Intermediate
Focus: Understanding + Platform navigation

CSA domains (high level)

Typical weight distribution:

Platform & Navigation
Instance Configuration
Self-Service & Automation
Database & Data Management
Reporting & Dashboards


This exam is very UI and concepts heavy.

ServiceNow CAD (Certified Application Developer)

This is the developer certification.
It proves you can actually build apps on ServiceNow.

Exam format
Questions: 60
Duration: 90 minutes
Passing score: ~70%
Question type:
Multiple choice
Multi-select
Scenario-based scripting questions

Difficulty level

Intermediate → Advanced
Focus: Real development and scripting

CAD domains (simplified)

Major focus areas:

App Engine & app creation
Client scripts
Business rules
Script Includes
GlideRecord
REST integrations
Scoped apps


This exam is code + logic heavy.

CSA vs CAD — Real difference
Area	CSA	CAD
Focus	Admin & configuration	Development & scripting
Difficulty	Easier	Harder
Coding	Almost none	Heavy JavaScript
UI navigation	Very important	Basic
Real scenarios	Moderate	High
Order to take	FIRST	AFTER CSA

Think like this:

CSA = Learn the platform
CAD = Build on the platform

Biggest beginner mistakes (VERY IMPORTANT)

These fail many students:

Mistake 1 — Memorizing without using a PDI

ServiceNow exams test where to click, not definitions.

You must practice in a Personal Developer Instance (PDI) daily.

Mistake 2 — Ignoring multi-select questions

Multi-select questions have zero partial marks.
If 3 answers are correct and you choose 2 → wrong.

Mistake 3 — Skipping small features

ServiceNow loves questions on:

Impersonation
Update sets
ACL basics
Form vs List behavior
Client vs Server scripts

Small topics = exam traps.

Mistake 4 — Jumping to CAD too early

Without strong CSA basics, CAD becomes very hard.

Mistake 5 — Not doing enough mock exams

Passing requires exam stamina + pattern familiarity.
